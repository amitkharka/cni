# Voice - UK News


## Installation

Use the npm package manager to install the application.

requires Node.js v12+ to run.

```bash
npm install
```

## Usage

```bash
npm start
```
To view the application navigating to your server address in your preferred browser.

```sh
localhost:8080
```

## Tech 
* JavaScript
* Node.js
* HTML5
* CSS3
